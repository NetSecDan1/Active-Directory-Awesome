# 12_JIRA_TEMPLATES — Ready-to-Use AD Jira Cards

> **Copy → Paste → Fill in the blanks.** These are standalone Jira card templates, not AI prompts. For AI-assisted generation, see `11_MASTER_AI_PROMPTS/06-jira-card-generator-prompts.md`.

## Templates

| File | When to Use |
|------|------------|
| [INCIDENT-template.md](INCIDENT-template.md) | Active AD outage or degradation |
| [CHANGE-REQUEST-template.md](CHANGE-REQUEST-template.md) | Any planned change to AD |
| [EPIC-template.md](EPIC-template.md) | Multi-week AD project |
| [SECURITY-FINDING-template.md](SECURITY-FINDING-template.md) | Vulnerability or security gap |
| [TASK-template.md](TASK-template.md) | Operational AD task |
| [PIR-template.md](PIR-template.md) | Post-incident review |

## Jira Label Standards

```
active-directory     — all AD-related work
entra-id             — Entra ID / Hybrid Identity work
identity-security    — security-focused identity work
p0 / p1 / p2 / p3   — incident severity
change-request       — all CAB-bound changes
runbook              — runbook creation/update work
tech-debt            — known technical debt items
```
