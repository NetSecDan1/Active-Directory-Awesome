> **Purpose**: World-class AI prompts purpose-built for Active Directory and Identity engineering. These prompts are used by senior AD architects, Microsoft CSS engineers, and elite incident commanders who leverage AI as a force multiplier.

---

## How to Use This Library

1. **Copy a prompt** into your AI chat (Claude, GPT-4, Gemini) as a system prompt or first message
2. **Activate the persona** that fits your current task
3. **Follow the output format** defined in each prompt for maximum value
4. **Chain prompts together** for complex, multi-phase work

## Prompt Quality Standard

Every prompt in this library meets the following bar:

- **Expert-calibrated**: Outputs match what a Microsoft CSS Principal or AD MVP would produce
- **Safety-encoded**: Read-only operations clearly marked; destructive paths gated
- **Structured output**: Machine-readable or human-scannable format
- **Context-aware**: Handles both on-prem AD and hybrid Entra ID scenarios
- **Chain-ready**: Can be combined with other prompts for complex workflows
